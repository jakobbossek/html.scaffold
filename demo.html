<h1>Introducing html.scaffold</h1>

Building up a webproject from scratch mostly requires a load of the always same steps to build up a directory structure, reset CSS settings to force the same or at least a similar behavior throughout the wide area of different browser rendering engines. This repeating work is counterproductive and tedious.

html.scaffold facilitates the start of a webproject by providing a convenient basis file structure, containing all the CSS styles needed to ensure browser independent presentation. Consequently the developer can focus on the really important things, i.e., creating a fantastic web experience for the enduser without worrying about the hurdles that the developer is usually faced with.

Hint: this is young project and therefore very much work in progress
